# Proyecto Final — API REST + Firebase + JWT + Frontend

Este proyecto implementa una aplicación completa que permite administrar un catálogo de productos mediante una API REST segura, cumpliendo con todos los requerimientos del Proyecto Final de TechLab.

La solución incluye:
✅ CRUD de productos (Crear, Leer, Actualizar y Eliminar)  
✅ Autenticación con JWT
✅ Almacenamiento en Firestore (Firebase)
✅ Arquitectura escalable en capas  
✅ Protección de rutas  
✅ Manejo de errores 400 / 401 / 403 / 404 / 500  
✅ Frontend conectado (Admin + Usuarios con carrito)

Tecnologías utilizadas:

✅ Backend
- Node.js
- Express
- Firebase Admin (Firestore)
- JSON Web Token (JWT)
- CORS
- Body-parser
- Dotenv

✅ Frontend
- React + Vite
- React Router DOM
- Bootstrap
- Context API (Auth + Carrito)
- Fetch API
